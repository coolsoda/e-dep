# Mitigating Interviewer Bias in Multimodal Depression Detection: An Approach with Adversarial Learning and Contextual Positional Encoding

## Overview

This repository contains our work on multimodal depression detection, presented at EMNLP 2025. All the data used in this work is publicly accessible, with a detailed description in the paper.
